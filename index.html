About this project

This Threejs project was created by Ali Zaib to test the limits of 3D model visualization in most modern browsers. This allows for a glimpse into the future of what interactive experiences will be.

In this project I tried not to respect the limits that exist in terms of performance. Therefore, this experience can be very intense for most computers these days.

The 3D model was optimized in Blender from a purchased model. This is where I spend most of my time on these projects.

Hope you like it and if you want you can follow me on Twitter and YouTube for more projects using Blender and Threejs.
